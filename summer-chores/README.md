Summer-chores is a  JavaScript project that has three main parts:
The first part, callbackVersion.js uses callback functions to test if someone finishes their chores or falls asleep somewhere along the sequence. To run, type node callbackVersion.js in the terminal.

The second part, promiseVersion.js uses Promises to test if someone finishes their chores or falls asleep somewhere along the sequence. To run, type node promiseVersion.js in the terminal.

The third part, asyncAwaitVersion.js uses async/await to test if someone finishes their chores or falls asleep somewhere along the sequence. To run, type node asyncAwaitVersion.js in the terminal.
